## ระบบที่ 1: Auth — สมัครสมาชิก, Login, Profile

### 1.1 เปิดหน้า Register
- [ ] **ขั้นตอน:** เปิด URL → `/register`
- [ ] **สิ่งที่ควรเห็น:** ฟอร์มสมัครสมาชิก (ชื่อ, เบอร์, อีเมล, รหัสผ่าน, ยืนยันรหัสผ่าน)
- [ ] **ผล:** 🟢 / 🔴
- **URL:** https://pharmacare-1783398975-production.up.railway.app/register

### 1.2 สมัครสมาชิก — สำเร็จ
- [ ] **ขั้นตอน:** กรอกชื่อ="สมชาย ใจดี", เบอร์="081-234-5678", อีเมล="somchai@test.com", รหัส="Test1234", ยืนยัน="Test1234" → กด "สมัคร"
- [ ] **สิ่งที่ควรเห็น:** แจ้งเตือน "สมัครสมาชิกสำเร็จ" → redirect ไปหน้า Login (หรือเข้าสู่ระบบอัตโนมัติ)
- [ ] **ผล:** 🟢 / 🔴

### 1.3 สมัครสมาชิก — อีเมลซ้ำ
- [ ] **ขั้นตอน:** กรอกข้อมูลเดิมอีกครั้ง (อีเมลซ้ำ)
- [ ] **สิ่งที่ควรเห็น:** แจ้งเตือน "อีเมลนี้มีผู้ใช้งานแล้ว" หรือ error ที่เหมาะสม
- [ ] **ผล:** 🟢 / 🔴

### 1.4 สมัครสมาชิก — รหัสผ่านไม่ตรง
- [ ] **ขั้นตอน:** รหัส="Test1234", ยืนยัน="Test5678"
- [ ] **สิ่งที่ควรเห็น:** Error "รหัสผ่านไม่ตรงกัน" (Client-side validation ก่อนส่ง)
- [ ] **ผล:** 🟢 / 🔴

### 1.5 Login — สำเร็จ
- [ ] **ขั้นตอน:** เปิด `/login` → กรอกอีเมล="somchai@test.com", รหัส="Test1234" → กด "เข้าสู่ระบบ"
- [ ] **สิ่งที่ควรเห็น:** Login สำเร็จ → redirect ไปหน้าเดิม (หรือหน้าแรก)

### 1.6 Login — รหัสผ่านผิด

### 1.7 Login — อีเมลไม่มีในระบบ

### 1.8 แก้ไขโปรไฟล์
- [ ] **ขั้นตอน:** Login → กดเมนูบัญชี → ไป `/account/profile`
- [ ] **สิ่งที่ควรเห็น:** ข้อมูลชื่อ, เบอร์, อีเมล (จากตอนสมัคร)
- [ ] **ขั้นตอนต่อ:** แก้ชื่อเป็น "สมชาย รักดี" → กด "บันทึก"

## ระบบที่ 3: Cart System — ตะกร้าสินค้า

### 3.1 เพิ่มสินค้าลงตะกร้า (จาก Product Detail)
- [ ] **ขั้นตอน:** เปิด Product Detail → กด "➕ เพิ่มลงตะกร้า"
- [ ] **สิ่งที่ควรเห็น:** Toast/Alert "เพิ่มสินค้าเรียบร้อย" หรือ Badge ตะกร้าอัปเดต (+1)
- [ ] **ตรวจสอบ:** DevTools → Network → มี POST `/api/cart/add` status 200
- [ ] **ผล:** 🟢 / 🔴

### 3.2 เพิ่มสินค้าซ้ำ
- [ ] **ขั้นตอน:** กดเพิ่มสินค้าเดิมอีกครั้ง
- [ ] **สิ่งที่ควรเห็น:** Badge เพิ่มขึ้น หรือ Toast แจ้ง "มีสินค้านี้ในตะกร้าอยู่แล้ว"
- [ ] **ผล:** 🟢 / 🔴

### 3.3 เปิดหน้า Cart
- [ ] **ขั้นตอน:** กดไอคอนตะกร้า (มุมบนขวา) → ไป `/cart`
- [ ] **สิ่งที่ควรเห็น:**
  - [ ] รายการสินค้าที่เพิ่ม
  - [ ] ชื่อสินค้า
  - [ ] จำนวน (มีปุ่ม +/- หรือ input)
  - [ ] ราคาต่อหน่วย
  - [ ] ราคารวมต่อรายการ
  - [ ] ยอดรวมทั้งสิ้น (Subtotal)
  - [ ] ค่าจัดส่ง (ฟรีเมื่อ > 500 บาท หรือ 50 บาท)
  - [ ] ปุ่ม "📋 สั่งซื้อ"
- [ ] **ผล:** 🟢 / 🔴

### 3.4 ปรับจำนวนสินค้า
- [ ] **ขั้นตอน:** กด "+" หรือเปลี่ยนจำนวน
- [ ] **สิ่งที่ควรเห็น:** ราคารวมอัปเดต, ยอดรวมทั้งสิ้นอัปเดต

### 3.5 ลบสินค้าออกจากตะกร้า

- [ ] **สิ่งที่ควรเห็น:** สินค้าหายไป, ยอดรวมลดลง

### 3.6 ตะกร้าว่างเปล่า

- [ ] **สิ่งที่ควรเห็น:** "🛒 ตะกร้าว่าง" + ปุ่ม "เลือกสินค้า"

### 3.7 Session Persistence (ไม่ใช้ localStorage)
